# 序章 

- 


朝起きる。顔を洗い、準備をする。そして鏡を見る。良い感じだ。

「よし！」           

トップロードは頬を軽く叩き気合を入れて、外へ走りに出かける。これがトップロードの一日の始まりだった。トゥインクルシリーズを引退した今も、この習慣は続いていた。

朝の冷たい空気が頬を叩き、肺を満たして、まだ硬い体と寝ぼけた心を起こしていく。そうして起きた体を走らせていくと段々と、

何となくモヤモヤとするのだ。それは引退してからずっとそうだった。アドマイヤベガ、テイエムオペラオー、メイショウドトウの三人が引退した後も走り続けた一年間のことを思い出すのだ。






「黄昏(ラグナロク)を超えたお前は、今海中から脱しようとしている」
「ええっと。引退して新しい私になりつつあるってことでしょうか？」

トップロードはギムレットの語る言葉を理解するために勉強した知識から、ギムレットの真意を推測する。

「お前がギムレーへと至るためには、共にヴァルハラへと至った古き戦友と話すことが必要かもしれないな」

ギムレットは考えるように顎に手を当てながらトップロードへと語りかける。こう見えてというのは失礼かもしれないが、ギムレットは非常に鋭敏な感覚を持っており、周りをよく見ている。言葉が難解ではあるのだが……。それが分かっているトップロードは真剣にギムレットの言葉へ耳を傾ける。

「古き戦友ですか？」
「ああ……全てが終わった今だからこそ、今に至った己のオリジンを思い出す必要がある」
「オリジン……」


# 承

その日の夕方、トップロードはトレセン学園の練習用コースをスタンドに座って見つめていた。ぼんやりとターフを見つめていると、何だか吸い込まれそうだとトップロードは思った。

「ひゃあ！！」

「やぁ、トップロードさん」
「もう！オペラオーちゃん！」

「はーはっはっは！すまない。なんだか呆けているトップロードさんを見てついね」

軽やかにステップを踏み、笑いながらトップロードの隣に座るオペラオー。トップロードは頬を膨らませながらも、その所作の身軽さになんとなく目を奪われた。

「それで？ぼんやりとターフを見つめてどうしたんだい？」
「全くもう……」

頬を膨らませているトップロードを華麗に流しながら、質問するオペラオー

「んーっと。なんだかいうのは難しいんですが、本当に離れてしまったんだなって」
「トップロードさんがかい？」
「はい」

「それなら思い出してみるかい？」
「え？」
「ボクらの物語を」


# 転

あの頃は自分を保つのに必死だった。

「私はずっとあなたに追い付きたかった。トゥインクルシリーズで。」

トップロードはまるで眩しいものでも見るかのように目を細め、隣のオペラオーを見つめた。

「でもその機会は来なかった。」

そしてトップロードは目を瞑る。それをオペラオーは何も言わずに見つめていた。
しばらく2人して黙っていると

「だが人間は、負けるように造られてはいない」

トップロードが突然語り出す。

「打ち砕かれることはあっても、負けることはないんだ」

オペラオーは少しだけトップロードを見つめて、その言葉に乗る。

「考えるな。爺さん」

そしてニヤリと笑いながら答える。

「決めた通り進めばいい。来たら来たで、それはその時のことだ」

そう良い終えるとオペラオーは 答え合わせをする。

「『老人と海』」
「はい」

トップロードはこの作品を読み返して、気に入っていた。苦境に陥っても諦めない老人がどこか他人事とは思えなかった。

「オペラではないね」
「ちょっと迷走しちゃいまして」

トップロードはオペラオーの言うことを理解したいと時々勉強している。そのことをオペラオーに伝えたわけではないが、「オペラではない」とわざわざ言うと言うことは

トップロードがこの作品に出会ったのはオペラオーの言うことを理解したいと勉強を始めてしばらく経った頃だった。


「結局、G1は一つも勝てませんでしたけどね。」


「不思議なんです。私の中に何か別の感覚があるような……そんな感じがして。その感覚が、誰かにもう一度だけ勝利を捧げたかったと言っているような……そんな……」

「なんて……何言ってるんでしょうね？私……」


# 結


「これからは次の世代がやってくれるさ」
「そう……ですか……」

相変わらず、さらっと厳しいことを言うものだとトップロードは思った。

「本当に凄いよ……君は……」
「オペラオーちゃん？」

オペラオーは噛み締めるように呟くと、トップロードの肩に頭をもたげた。戸惑うトップロードをよそにオペラオーはゆっくりと目を閉じる。

「トップロードさんが走り続けてくれたおかげで、僕らの時代を示し続けてくれたおかげで、僕らの物語は素晴らしいエピローグを迎えることができたんだ」
「そんな、大したことは……」
「大したことさ。」

謙遜しようとするトップロードの言葉をわざとらしく遮る。

「そう……大したことだよ」

オペラオーは再び噛み締めるように呟いた。トップロードは寄り添ってきたその小さな体を感じながら考える。年間無敗、グランドスラム。当時のオペラオーにはこの小さな体にどれほどの重圧と責任がのしかかってきたんだろう、と。

「エピローグがある物語は、強く鮮烈に観客の記憶に残るのさ」

トップロードはその言葉を聞いて、思い出す一節があった。そしてその一節を誦じる。

「この世は舞台なり。誰もがそこでは一役演じなくてはならない」
「『お気に召すまま』」

すぐに引用された作品を答えるオペラオー。その声色は笑っていた。

「トップロードさんが自分の役を君自身で見出せて良かった」
「私も、期待に応えられたのなら良かった」

トップロードは肩に乗るオペラオーの頭を優しく撫でた。

「ありがとう……トップロードさん」

オペラオーのその言葉を聞いてトップロードは思い出した。最後の有馬記念の記憶を。

『トップロード！ありがとう！』

あの日、ターフの上を去る時にファンの人から言われた言葉。なぜ『ありがとう』だったんだろう？ずっとそう思っていた。でもようやくわかった気がした。

(そっか……私はこの為に……)

この世はすべて　ひとつの舞台、
男も女も　人はみな役者に過ぎぬ。

もしもその言葉が本当で、長く走った自分に役割があったとすれば、誰かの記憶に刻まれた自分の走りが、誰かを勇気づけて、そしてその輪が広がっていく。

だから、自分はもう一年頑張ろうと決めたのかもしれない。トップロードは自分の中にいる誰かも同じ気持ちであることを願った。

「どういたしまして」

オペラオーの静かな呼吸と温もりを感じながら、しばらく学園のターフを見る。するとすぅすぅとオペラオーが寝息を立て始める。それを見たトップロードはオペラオーの腰を倒れないように支えた。

(オペラオーちゃんのこんな姿、初めて見たかも……)

トップロードはそう思いながらオペラオーの温もりを感じつつ、ぼうっと遠くを眺めていた。

「オペラオーさん、凄く気持ち良さそうに寝てますねぇ。」
「珍しいわね。」
「ふふっ、こうして見てるとオペラオーちゃんは年下なんだって改めて思いますね。」

「少しは人生の先輩らしいこと出来たのかな……」
「「……」」

アヤベは呆れたような顔で、ドトウは困ったような顔でトップロードを見つめている。

「な、なんですか？」
「貴方のそういうところ、好ましいと思うけど少しは直した方がいいと思うわ。」
「え？」
「オ、オペラオーさんはトップロードさんのこと凄く褒めてました。だから、えっと、もっと自信を持った方がいいと思います！」
「え、えぇ！？」
